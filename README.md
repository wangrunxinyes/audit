You install it like this:
```
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/balrok/yii2-audit"
        }
    ],
    "require": {
        "monolog/yii2-audit": "dev-balrok-bugfix"
    }
}
```
